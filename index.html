Data Driven Eye Candy

Animations are a great way to waste time

There are many ways to use D3 in terms of displaying massive amounts of important data in a visually appeasing and understandable way.

The ways objects are animated in D3 is similar to how it is done in video games. At the core of this beautiful framework we find advanced ways of rendering things such as lighting, object animation and effects relative to user interaction.

Having this type of ability within web development is quite remarkable and should be fully utilized whenever possible, however, it can also be used just to make a site more dynamic and less boring.
